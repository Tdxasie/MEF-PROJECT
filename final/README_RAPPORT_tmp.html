DOMAINE 1: carré [0,1]x[0,1]

  • -delta u = fomega

    1. u(x,y) = 16xy(1-x)(1-y) -> fomega = 32(x(1-x)+y(1-y))

    2. u(x,y) = sin(PIx)sin(PIy) -> fomega = 2*PI²*sin(PIx)*sin(PI*y)

  • -delta u + u = fomega

    1. u(x,y) = cos(PIx)cos(PIy) -> fomega = (1+PI²)*cos(PI*x)*cos(PI*y)
